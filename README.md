# SivanZeroX Public Issues

This repository is the public issue tracker for products maintained under the `SivanZeroX` account.

## 中文说明

这个仓库用于公开收集问题反馈，不包含核心源码。

- 提交 Bug：请在 Issues 中提供复现步骤、预期结果、实际结果、版本号、系统信息
- 功能建议：请说明使用场景、目标用户、预期收益
- 讨论类问题：优先给出你已经尝试过的方法和结果

建议标题格式：

- `[Bug] 简要问题描述`
- `[Feature] 简要需求描述`
- `[Question] 简要问题`

## English

This repo is used for public feedback only. Core source code is not hosted here.

- Bug reports: include steps to reproduce, expected behavior, actual behavior, app/extension version, and environment
- Feature requests: include use case, target users, and expected impact
- Questions: include what you have already tried

Suggested title format:

- `[Bug] short summary`
- `[Feature] short summary`
- `[Question] short summary`

## Scope

- Issue collection and discussion
- Feedback triage and status tracking
- Release-related bug follow-up

## Not in this repository

- Core business code
- Private implementation details
- Internal deployment/configuration assets
